# Mocha Green
A dark catppuccin rice with bright neon green accent color, designed with both looks and usability in mind. 

This config also try to be as minimalist as possibile, without sacrificing too much on functionality.

## Features
### Desktop Environment (X11)
1. Qtile WM
1. Polybar
1. Rofi launcher
1. Dunst notification
1. Redshift screen temperature
1. Picom compositor
1. Unified GTK & QT (gtk2) theming
1. Nitrogen wallpaper
1. Lightdm GTK
1. Lxsession polkit
1. Xscreensaver with flying toasters
1. ZSH with syntax highlighting

### Utilities
There may not be any special config files for some of these applications, but I'd recommend using them for this rice.

1. `neovim` or `vim` text editors
1. `code` OSS build of Vscode
1. `kitty` terminal
1. `nemo` file manager
1. `btop` task manager
1. `slimbookbattery` power manager
1. `qt5ct` for QT configuration
1. `lxappearance` for GTK configuration
1. `mpv` video player
1. `gpicview` image viewer
1. `rhythmbox` or `lxmusic` music player
1. `librewolf` or `brave` web browser
1. `keepassxc` password manager
1. `kdeconnect` really good sync app
1. `pcmanfm` lightweight file manager
1. `qalculate-gtk` scientific calculator
